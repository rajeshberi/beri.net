# beri.net - THE D*AI*LY BRIEF

Official website for THE D*AI*LY BRIEF newsletter by Rajesh Beri.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Hosting:** Vercel
- **Domain:** beri.net (Cloudflare DNS)

## Development

```bash
pnpm install
pnpm dev      # http://localhost:3000
```

## Build

```bash
pnpm build
pnpm start    # Production server
```

## Deploy

```bash
vercel --prod
```

## Features

- ✅ Responsive design
- ✅ Newsletter signup form (ready for Beehiiv integration)
- ✅ Social links (LinkedIn, Twitter/X)
- ✅ SEO-optimized
- ✅ Fast static generation
